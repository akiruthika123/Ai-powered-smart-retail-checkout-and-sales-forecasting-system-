🛍️ Smart Textile Retail: AI-Powered Inventory & Pricing System
An advanced Retail Intelligence System designed for independent textile businesses. This project uses Ensemble Learning (Random Forest) to automate pricing predictions and inventory alerts, bridging the gap between traditional retail and 2026 data science standards.

🚀 Key Features
Predictive Pricing Engine: Uses machine learning to suggest optimal Selling Prices based on Buying Costs and item categories.

Smart Inventory Alerts: Automated "Restock" triggers based on real-time availability data.

Data Consistency Pipeline: Built-in preprocessing to handle inconsistent naming and missing values in retail datasets.

Advanced Analytics: Visualizes top-performing categories and price distributions for better stakeholder decision-making.
